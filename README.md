# Distributed Systems Web Interface

A unified web-based platform demonstrating core Distributed Systems concepts
using Python (Flask), Java, and WebSockets.

## Features
- Socket Programming (Client-Server)
- RPC Services (Tax, CGPA, Voting)
- P2P File Sharing
- Shared Memory Simulation
- Real-time WebSocket Chat
- Java RMI File Upload & Download
- Unified Web Interface

## Tech Stack
- Python (Flask, Flask-SocketIO)
- Java (RMI)
- HTML, CSS, JavaScript
